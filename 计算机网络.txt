	法定标准（由权威机构制定的正式的，合法的标准）：OSI
	事实标准（某些公司的产品在竞争中占据了主流，时间长了，某些产品中的协议和技术就成了标准）：TCP/IP
	RFC：因特网标准的形式
		因特网草案：该阶段不是RFC文档
		建议标准：从该阶段开始成为RFC标准
		草案标准：拿到IETF，IAB审核（已取消）
		因特网标准
	相关组织
		ISO（建立了ISO参考模型，HDLC协议）
		国际电信联盟ITU（制定了通信规则）
		国际电器电子工程师协会IEEE（学术机构，IEEE802系列标准，5G）
		internet工程任务IETF（负责因特网相关标准的制定）

计算机网络的性能指标{
	速率：数据率或数据传输率或比特率（每s传输多少比特）
		速率的进制{一般速率的单位用小写
			1kb/s = 10^3b/s
			1Mb/s = 10^3kb/s = 10^6b/s
			1Gb/s = 10^3Mb/s = 10^6kb/s = 10^9b/s
			1Tb/s = 10^3Gb/s = 10^6Mb/s = 10^9kb/s = 10^12b/s
		}
		储存容量的进制{一般容量的大小单位用大写
			1Byte = 8Bit
			1KB = 1024B
			1MB = 1024KB
			1GB = 1024MB
			1TB = 1024GB
		}
	带宽：原指某个信号具有的频带宽度，即最高频率与最低频率之差，单位是赫兹（HZ）
		计算机网络中，带宽用来表示网络的通信线路传送数据的能力，通常是单位时间内从网络中的某一点到另一点所能通过的“最高数据率”。单位是“比特每秒”，b/s,kb/s,Mb/s,Gb/s，（可以理解为网络设备支持的最高的速度,是网络入口发送的速率，而不是信道上传输的速率）
	吞吐量：单位时间内，通过某个网络（信道，接口）的数据量（接收和发送的数据量的总和），单位b/s,kb/s,Mb/s等
		吞吐量受网络的带宽或网络的额定速率的限制

	时延：数据（报文/分组/比特流）从网络的一端传送到另一端所需的时间。也叫延迟或迟延，单位是s
		发送时延（传输时延）：从发送分组的第一个bit算起，到该分组的最后一个bit结束所需的时间
			发送时延 = 数据长度 / 信道带宽（发送速率）
		传播时延：电磁波传输一定距离所需的时间，取决于电磁波的传播速率和链路长度
			传播时延 = 信道长度/ 电磁波在信道上的传播速率
		排队时延：等待输出/输入链路可用（例如排队等待路由器处理）
		处理时延：处理数据所用的时间（路由器等）
		高速链路提高的是发送速率，减小了发送时延，如果采用不同的介质可以改变电磁波的传输速率，进而可以改变传播时延
	时延带宽积：（以比特为单位的链路长度，即“某链路现在有多少比特”）
		时延带宽积 = 传播速率 × 带宽
		单位：bit		s		 b/s
	往返时延RTT：（从发送方发送数据开始，带发送方收到接受到接受方的确认，及发出数据的第一个bit开始到接收到确认信息的第一个bit结束）
		RTT越大，在收到确认前，可以发送的数据越多
		RTT包括（不包括传输（发送）时延）
			往返时延： 传播时延 × 2
			末端处理时间：接受方收到发送方的数据并对数据进行处理之后发出确认信息
	利用率：
		信道利用率：
			链路有多长时间是有数据的
			信道利用率 = 有数据通过的时间 / （有 + 无）数据通过的时间
		网络利用率：
			信道利用率加权平均值

分层结构{
	为什么要分层：{
		发送文件的步骤：
			发送前激活数据链路
			告诉网络如何识别目的主机
			发起通信的主机要查明目的主机是否开始并正常连接到了网络
			发起通信的计算机要弄清目标主机的文件管理程序是否做好准备（是否可以接收，是否有足够的空间等）
			确保差错和意外可以解决
			…………
		发送文件等请求需要处理的东西繁多复杂，不便于管理，分层可以条理清晰地解决这些问题（将大问题转化为小问题）
	}

	实体：每一层当中的活动元素,第n层中的活动元素成为n层实体
	对等实体：同一层的实体
	协议：用以协助进行中网络上的对等实体之间进行数据交换的规定，要求叫做协议，协议是水平方向的，只有对等实体之间才存在协，
		协议的三要素：
			语法：规定传输数据的格式
			语义：规定所要完成的功能
			同步：规定各种操作的顺序
	接口：上层使用下层服务的入口，计算机网络中的接口是逻辑性的
	服务：下层为上层提供的功能调用，计算机网络中上层使用下层服务，下层为上层提供服务
	分层的基本原则：
		各个层次之间相互独立，每层只实现一种相对独立的功能
		每层之间的界限自然清晰，便于理解，尽量减少不同层次之间的交流
		结构上可分割，每层用最适合的技术实现
		保持下层对上层的独立性，上层单向使用下层提供的服务
		整个分层结构应该能促进标准化工作
	SDU数据服务单元：为完成用户所要求的功能所需传输的数据
	PCI协议控制信息：控制协议操作的信息
	PDU协议数据单元：对等层次之间传送的数据单位
	网络结构是从功能上描述的
	计算机网络体系结构简称网络体系结构是分层结构
	每层通过相应的协议完成该层的工作
	计算机网络体系结构是计算机网络各层及其协议的集合
}
